# Logo options v2 — sharper direction

## Critical takeaway

Earlier options (A–E) looked soft / decorative because **raster AI logos** often:

- add fake 3D, glow, uneven stroke weights
- fail at **16px favicon**
- feel like stickers, not enterprise marks

**What looks best for Epiow (B2B Organization OS):**

1. **One mark only** — no dual metaphors fighting
2. **Flat solid color field** (indigo/navy) + pure white mark
3. **Readable at 16px** — test by squinting
4. **Meaning in letter construction**, not illustration (no orbit solar systems, no hex tech clutter)
5. **Ship as SVG**, not AI PNG

Comparables that win: Linear, Stripe, Notion, Feishu — **simple geometry**, not cinematic icons.
