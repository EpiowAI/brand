# Direction W3 — fix “幼 / 窄 / 一忽一忽”

## What was wrong with W1–W6

- Letters built from **separate bars/rects** → felt like UI kit pieces, not type
- Stroke too light, proportions narrow → not enterprise-dignified

## What W3 does

| Feedback | Fix |
|----------|-----|
| 唔得體 | Real **Bold grotesque** fonts → outlined SVG paths |
| 好幼好窄 | Bold masters; tracked variants; heavier continuous customs |
| 一忽一忽 | Continuous glyph outlines (not 4-bar E) |

## Look first

1. **T3 / T4** Liberation Sans Bold (+ tracking)
2. **T1 / T2** DejaVu Sans Bold
3. **L2** lockup (icon + wordmark)
4. C-series only if you still want custom geometric

`options-w/` W1–W6 are superseded for critique.
