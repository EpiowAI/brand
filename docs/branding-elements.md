Each Sylphx portfolio company brand repo maps to these layers.

## Industry pattern (research summary)

Typical brand guideline sections:

1. Introduction & story  
2. Strategy / positioning  
3. Verbal identity  
4. Logo (construction, clear space, min size, co-branding, misuse)  
5. Color (primary/secondary/neutral + codes)  
6. Typography  
7. Brand assets (patterns, icons, photography)  
8. Layout / grid  
9. Voice  
10. Application examples  

Sources informing this structure: professional brand-guide practice (logo construction grids, clear-space measured from a letter unit, multi-format color codes, story-led books).
